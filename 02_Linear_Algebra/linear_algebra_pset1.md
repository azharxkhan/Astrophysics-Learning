# Linear Algebra - Problem Set 1

## Problem 1 - Solving linear systems
Solve the system:
2x + 3y = 5
4x - y = 1

Use both substitution and matrix methods.

---

## Problem 2 - Eigenvalues of a 2x2 matrix
Given the matrix
A = [[2, 1],
     [1, 2]]

1. Find its eigenvalues.
2. Find eigenvectors corresponding to each eigenvalue.
3. Interpret what this matrix does geometrically to vectors.

---

## Problem 3 - Rotation matrix
The 2D rotation matrix by angle θ is:
R(θ) = [[cosθ, -sinθ],
        [sinθ, cosθ]]

1. Show that R(θ1) * R(θ2) = R(θ1 + θ2).
2. Show that R(θ)^T * R(θ) = I.

---

## Problem 4 - Change of basis
Given two basis vectors in R^2:
e1 = (1, 1), e2 = (1, -1).

1. Write the matrix that transforms coordinates from the standard basis to the {e1, e2} basis.
2. Compute its inverse.

---

## Problem 5 - Normal modes (conceptual)
Consider two identical masses m connected by three springs in a line with equal spring constant k and fixed ends.

1. Set up the equations of motion.
2. Express them in matrix form.
3. Explain how eigenvalues of this matrix relate to normal mode frequencies.
