# Advanced Astrophysics - Problem Set 1

## Problem 1 - Hydrostatic equilibrium
Write the differential equation for hydrostatic equilibrium inside a star. Explain the physical meaning of each term.

---

## Problem 2 - Radiative transfer
Write the basic radiative transfer equation in a plane parallel atmosphere. Define intensity, source function, and optical depth.

---

## Problem 3 - Polytropic equation of state
Explain what a polytropic equation of state is and write the relation between pressure and density for a polytrope. Give examples of astrophysical objects that can be approximated as polytropes.

---

## Problem 4 - Chandrasekhar mass (conceptual)
Describe qualitatively how degeneracy pressure leads to an upper mass limit for white dwarfs. State the approximate value of the Chandrasekhar mass.

---

## Problem 5 - Friedmann equations
Write the Friedmann equation for the scale factor a(t) in a flat universe with matter and cosmological constant. Explain the meaning of each term.
